# 🏫 Smada Campus Invation - Kampus Ekspo

Landing page interaktif dan responsif yang dirancang khusus untuk acara **Smada Campus Invation (SCI)**. Website ini berfungsi sebagai media informasi bagi calon mahasiswa untuk menjelajahi profil **Politeknik Elektronika Negeri Surabaya (PENS)**, mulai dari departemen, detail program studi, prospek kerja, hingga kontak alumni.

## ✨ Fitur Utama
* **Desain Premium & Modern**: Menggunakan tema *luxurious deep purple* dengan sentuhan *glassmorphism* dan aksen emas.
* **Modals Informasi Interaktif**: Detail program studi dan prospek lulusan di setiap departemen (DTIK, DTE, DTMK, DTME) dapat diakses secara dinamis tanpa berpindah halaman.
* **Infinite Alumni Carousel**: Menampilkan kartu profil alumni SMADA yang menempuh pendidikan di PENS dengan animasi *marquee/carousel* yang halus.
* **Animasi Imersif**: Didukung oleh `AOS (Animate On Scroll)` dan `Animate.css` untuk transisi elemen visual yang interaktif.
* **Fully Responsive**: Optimal diakses dari perangkat *mobile* (smartphone), tablet, hingga desktop/laptop.

## 🛠️ Teknologi yang Digunakan
* **HTML5** – Struktur semantik halaman web.
* **CSS3** – Kustomisasi variabel global (`:root`), tata letak (*Flexbox* & *Grid*), serta efek transisi modern.
* **Vanilla JavaScript** – Logika manipulasi DOM untuk navigasi *mobile*, penutupan modal otomatis (*click outside*), dan pengisian konten prodi secara dinamis.
* **Library Pihak Ketiga**:
    * [Feather Icons](https://feathericons.com/) – Untuk ikon minimalis.
    * [Animate.css](https://animate.style/) – Untuk efek animasi *entrance* pada *hero section*.
    * [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/) – Untuk memicu animasi saat halaman di-scroll.

## 📂 Struktur Variabel CSS Utama (`style.css`)
Project ini menggunakan skema warna terpusat untuk kemudahan kustomisasi:
* `--bg`: `#1c093a` (Deep Purple)
* `--primary`: `#ffd24d` (Bright Gold)
* `--second`: `#29b6bd` (Vibrant Cyan)
* `--glass-bg`: `rgba(20, 9, 39, 0.75)`
